 ChronometerOnline
simple online choronometer made on razor with mvc, easy to use.

The interface was designed to be simple and clean, organazing a:

- Centered layout  
- Styled buttons with different colors  
- Scrollable lap list 
- Minimalist and clean header

-  Features

   Start the chronometer  
  Pause the chronometer  
   Reset the time  
   Record lap times  
  simple and modern user interface

the project follows a structure inspired in MVC (Model-View-Controller):

- Model  
  Handles the chronometer data and status.

- Service controls the logic 
  Contains the logic and interactions.  

- View Blazor   
  Implements the user web interface using Razor.
